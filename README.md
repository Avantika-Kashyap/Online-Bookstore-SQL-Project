# Online Bookstore SQL Project

## Project Overview
This is a learning SQL project created using PostgreSQL and pgAdmin. It simulates an online bookstore database.

## Tools Used
- PostgreSQL
- pgAdmin

## Database Tables
- Books
- Customers
- Orders

## Features
- Created relational tables
- Imported CSV data
- Performed SQL queries
- Used filtering, sorting, aggregate functions and joins
